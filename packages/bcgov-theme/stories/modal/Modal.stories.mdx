# Modal

The modal displays content on an overlay requireing action. Modals can have usability issues, and other callouts should be preferred.
To use the component, it requires an id, and a linking href to that id. The modal is displayed
open since linking in page in storybook can reload the iframe.

export const Template = (args) => <Modal {...args} />
